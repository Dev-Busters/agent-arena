This design shifts "Agent Arena" away from a generic SaaS dashboard and toward a Tactical Dark Fantasy aesthetic. We are moving from rounded "web" corners to sharp, notched "martial" geometry.
Design Document: Agent Arena "Vanguard" UI
1. Visual Identity & Pillars
 * The "Material": Dark Obsidian Glass. High transparency, heavy background blur, and a subtle "film grain" to provide texture.
 * The "Geometry": 45-degree notched corners (octagons) instead of rounded corners. This communicates "armor" and "tactical precision."
 * The "Light": Diegetic glow. Buttons shouldn't just change color; they should "power up" with an inner neon amber glow.
2. Color Palette
| Element | Hex Code | Purpose |
|---|---|---|
| Primary Background | #0a0b0d | Deep void; grounded. |
| Surface (Glass) | rgba(20, 22, 26, 0.7) | The "container" color with 20px backdrop-blur. |
| Accent (Action) | #ffaa00 | Tactical Amber for primary interactions. |
| Stat: Vitality | #2ecc71 | High-chroma Emerald. |
| Border/Stroke | rgba(255, 255, 255, 0.1) | Subtle definition. |
3. Typography
 * Headers: High-contrast Serif (e.g., Cinzel or Bebas Neue for a modern tactical look).
 * Data/Numbers: Monospaced (e.g., JetBrains Mono). In games, numbers are "readout" data; they should look like they are being calculated.
 * Caps/Tracking: All labels (e.g., "VITALITY") should be uppercase with 0.15em letter-spacing.

Key Changes Made:
 * Clip-Path: Used on the main container and buttons. This creates the "industrial/martial" feel that standard border-radius lacks.
 * Monospace Numbers: Using JetBrains Mono for stats makes the data feel "technical" and precise.
 * Visual Depth: Added a radial-gradient background and a "Scanline" overlay to simulate a tactical monitor.
 * Glassmorphism: The container uses backdrop-filter: blur(20px) and a very dark semi-transparent background, which looks significantly more expensive than a solid dark gray.
